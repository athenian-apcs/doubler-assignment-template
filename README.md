# Doubler Assignment 

Add code to [MyClass.doubleIt()](src/main/java/Doubler.java) to return a doubled value.

## Run your code with:
Go to MyClass and click on `Run` above the `main()` method.

## Run your tests with:
```shell script
./gradlew clean test
```

## Clean the build directory with:
```shell script
./gradlew clean
```
